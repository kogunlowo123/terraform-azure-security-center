Microsoft Azure Terraform Module Security

terraform-azure-security-center

Executive Summary

Production-ready Terraform module for deploying Microsoft Defender for Cloud at subscription level with security contacts, per-resource Defender plans (VMs, App Services, SQL, Storage, Key Vault, Containers, Cosmos DB, ARM, DNS, Cloud Posture), auto-provisioning of the Log Analytics agent, MCAS and Defender for Endpoint integrations, Sentinel onboarding, custom security assessment policies, and automated security workflow triggers via Logic Apps.

1. Defender plans monitor Azure resources at the subscription level. 2. Threat detection generates security alerts and recommendations. 3. Custom assessments evaluate compliance posture. 4. Security automations trigger Logic App workflows for high-severity alerts. 5. Security contacts receive notifications. 6. All data flows to Log Analytics and optionally to Sentinel for SIEM.

Security Controls

Threat Protection

- Per-resource Defender plans (14 resource types)
- Standard tier for advanced threat detection
- Subplan support (P1/P2 for VMs)
- Vulnerability assessment auto-provisioning

Integration

- MCAS for shadow IT discovery
- Defender for Endpoint (WDATP/MDE)
- Sentinel onboarding for SIEM
- Log Analytics workspace assignment

Compliance

- Custom security assessment policies
- Severity classification (High/Medium/Low)
- Threat categorization
- Implementation effort tracking

Incident Response

- Security contact notifications
- Admin alert forwarding
- Automated workflows via Logic Apps
- Event source filtering (alerts, compliance)

Industry Adaptation

Enterprise

All Defender plans at Standard tier, MCAS + MDE integration, Sentinel for centralized SIEM, automated incident response

Financial Services

SQL Server and Storage Defender plans, custom assessments for encryption and MFA, compliance automation workflows

Healthcare

VM vulnerability assessment, Key Vault protection, custom logging assessments, auto-provisioning for all servers

Government

Cloud Posture Management (CSPM), ARM/DNS protection, full Sentinel integration, custom security assessment policies

Production Readiness Checklist

  • Security contact configured with email and admin alerts
  • Defender plans enabled for all critical resource types
  • Auto-provisioning of Log Analytics agent enabled
  • MCAS and MDE integrations activated
  • Log Analytics workspace assigned
  • Enable Sentinel onboarding for SIEM correlation
  • Create custom security assessments for org policies
  • Set up security automations for high-severity alerts
  • Enable vulnerability assessment auto-provisioning
  • Review and tune Defender plan subplans (P1/P2)
  • Configure Logic App workflows for incident response
