# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-15

### Added

- Security contact configuration with email, phone, and notification settings
- Microsoft Defender for Cloud pricing tiers for 14 resource types
- Subplan support for VirtualMachines (P1/P2) and StorageAccounts (DefenderForStorageV2)
- Auto-provisioning of Log Analytics agent
- Log Analytics workspace assignment at subscription scope
- MCAS (Microsoft Cloud App Security) integration
- WDATP/MDE (Defender for Endpoint) integration
- Sentinel onboarding setting
- Custom security assessment policies with severity, categories, and threats
- Security automation workflows with event sources and rule-based filtering
- Logic App action integration for alert forwarding
- Regulatory compliance assessment change detection
- Comprehensive input validation for email, resource types, and severity levels
- Basic, advanced, and complete usage examples
